This is a web application that demonstrates containerization using Podman. Follow the steps below to build, run, and manage your containerized application.

üîß Setup Instructions

1 Create the Dockerfile:
FROM nginx:alpine COPY index.html /usr/share/nginx/html/index.html
2 Build the image:
podman build -t helloworld-webapp .
3 Run the container:
podman run -d --name "HelloWorldWebAppContainer" -p 8080:80 helloworld-webapp
